Default timezone example. Here timezone property is a object which has two properties they are location and offset in minutes.

Dialog can be styled using below custom properties to change dialog background color or pointer color etc.

Timezone with format-time in minutes.Format-type is a property which is used for dislpaying timezone offset as a suffix either in hours or minutes. By default it is displayed in hours format. For example if offset is +330 minutes then it is in hours format +5.5 and in minutes format 330.

Timezone with responsive-width which is property responsible for changing the view depending on the browser width. If browser width matches or less than responsive width then paper-timezone will be displayed as dropdown with search functionality.
